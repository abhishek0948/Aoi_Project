### Simple Schema Overview

**Core Entities:**

1. **AOIFeature** (Area of Interest)
   - Primary entity representing user-drawn shapes
   - Persisted in browser localStorage
   - Can be Point, Polygon, or Rectangle
   - Auto-generates ID, timestamp, random color
   - Calculates area for polygons/rectangles

2. **Layer** (Map Base Layer)
   - Represents map tile layers (satellite, street)
   - Only one active at a time
   - WMS protocol for NRW satellite imagery
   - Managed in React state (not persisted)

3. **GeocodingResult** (Search Result)
   - Temporary data from Nominatim API
   - Used for location search
   - Converted to map bounds for navigation
   - Not persisted
